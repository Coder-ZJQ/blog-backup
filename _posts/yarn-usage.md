### 初始化一个新项目
``` bash
$ yarn init
```

### 添加依赖包
``` bash
$ yarn add [package]
$ yarn add [package]@[version]
$ yarn add [package]@[tag]
```

<!-- more -->

### 更新依赖包

``` bash
$ yarn upgrade [package]
$ yarn upgrade [package]@[version]
$ yarn upgrade [package]@[tag]
```

### 移除依赖包
``` bash
$ yarn remove [package]
```

### 安装项目的所有依赖包
``` bash
$ yarn
```
***or***
``` bash
$ yarn install
```
